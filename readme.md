## 📌 Project Overview

Flight delays are a major challenge in the aviation industry, affecting airline operations, airport management, and passenger experience. Predicting potential flight delays can help airlines and airports take preventive actions, optimize scheduling, and improve overall efficiency.

This project develops a **machine learning-based system to predict whether a flight will be delayed or on time** using historical airline data.

The project follows a complete **end-to-end data science pipeline**, including:

- Data preprocessing and cleaning  
- Exploratory data analysis (EDA)  
- Feature engineering  
- Machine learning model training  
- Model evaluation  
- Deployment using a Streamlit web application

---

## 🎯 Project Objectives

The main objectives of this project are:

- Analyze historical flight delay data to identify key patterns and factors contributing to delays.
- Build machine learning models capable of predicting flight delays.
- Evaluate multiple models to determine the best-performing approach.
- Deploy the trained model through an interactive web application.

---

## 📊 Dataset Information

The dataset used in this project contains historical airline flight data including:

- Airline carrier information
- Departure and arrival schedules
- Flight distance
- Delay causes such as weather and air traffic
- Operational flight data

Key features used in the model include:

- Month
- Day of Week
- Departure Hour
- Arrival Hour
- Airline Carrier
- Origin Airport
- Destination Airport
- Distance
- Weather Delay
- Air Traffic Delay
- Late Aircraft Delay

---

## 🧠 Machine Learning Approach

Several machine learning models were trained and evaluated:

- Logistic Regression
- Decision Tree
- Random Forest Classifier

The models were evaluated using performance metrics such as:

- Accuracy
- Confusion Matrix
- Feature Importance

The **Random Forest model** was selected as the final model due to its strong predictive performance.

---

## 🌐 Deployment

The trained model is deployed using **Streamlit**, allowing users to interact with the prediction system through a web interface.

Users can input flight information and receive a prediction indicating whether the flight is likely to be delayed.
